# PROSTARS™

A children's development company for Canterbury, New Zealand. Founder: Elanza Muller.

PROSTARS teaches children physical confidence and inner readiness — presence, resilience, observation, social connection, and the courage to try — through movement and play, with visible weekly proof for parents. It is a fresh, ownable brand: not a Playball re-run, not a re-skin of the earlier `Proteus` prototype.
